Airport Management System (Console-Based Application)
 Project Overview

The Airport Management System is a console-based application developed in C++ that simulates basic airport operations.
It provides different functionalities for Passengers, Staff Members, and Air Hostesses within a single integrated system.

This project demonstrates the use of:
Object-Oriented Programming (OOP)
File Handling
Structured Programming
Menu-driven Console Interface

Features
Passenger Module
Book flight tickets
View booking details
Check available flights
Enter passenger information

Staff Module
View salary details
Check assigned responsibilities
Access staff-related records

Air Hostess Module
View available flights
Check flight assignments
Identify flights departing from current airport
Manage work schedule details

Technologies Used
Language: C++
Concepts Used:
Classes and Objects
Functions
File Handling
Conditional Statements
Loops
Menu-driven programming
